# Cuemby landing page

A bespoke, single-page brief prepared for Angel Ramirez and Hitomi Mizugaki at eMerge Americas 2026 (Booth SS14), built in Cuemby's own visual vocabulary (Montserrat, navy + cyan-teal, indigo accents, chip/tag treatment) so that it reads as a native extension of cuemby.com rather than an Eduba pitch. The page frames the fit as a standing LATAM partnership (Cuemby runs the Kubernetes layer, Eduba runs the methodology layer above it), cites the ICM paper as the methodology the relationship rests on, carries the enablement proof (Correlation One and KPMG UK), and closes on a 30-minute conversation with Matt Creamer, Eduba's CRO. No Eduba branding appears except a single footer line.

Intended URL path: /for/cuemby
